# Web_Astra

## Astra

アニメのレビュー・評価・配信状況・スレッドを

ひとまとめで確認できる Web アプリ

---

## Astra とは

Astra は、アニメ作品について

- どんな評価なのか
- 他の人はどんな感想を持っているのか
- どこで配信されているのか

を **ひとまとめで確認できる Web アプリ**です。

レビューサイト・掲示板・配信検索を行き来しなくても、

Astra だけでアニメの情報と声をまとめて見ることができます。

---

## なぜ作ったか

アニメを見る前や見終わったあとに、

- 他の人の率直な感想を知りたい
- 評価の雰囲気をざっくり掴みたい

と思うことが多くありました。

しかし、

- 掲示板は少し重い
- SNSは話題が流れやすい
- レビューサイトは堅い

と感じることもあり、

**もっと気軽に、同じアニメを見ている人同士で感想を共有できる場所**

があればいいと思い、このアプリを作りました。

---

## できること

### アニメ作品の情報を確認

- 作品ごとの基本情報を一覧・詳細で表示

### レビューを見る／書く

- ログインするとレビュー投稿が可能
- 他の人の感想や評価をすぐに確認できます

### 配信状況のチェック

- どの配信サービスで視聴できるかを表示

### 2ch風スレッド機能

- アニメごとにスレッドを作成可能
- 匿名感覚で気軽に書き込み
- リアルタイムに近い感覚で感想を共有
- 掲示板ほど堅くなく、SNSほど流れない設計

### Googleログイン

- Googleアカウントで簡単にログインできます
